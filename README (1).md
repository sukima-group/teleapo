## オートコールについて（重要）

スマホブラウザ（PWA）の制約上、ネイティブアプリ（TFT等）のような「完全無人の自動連続発信」は
OSの権限が必要で実装できません。本システムは現実的な最善策として「**半自動連続発信**」を実装:

- 応答なしは自動でスキップ（次の番号へ）
- 通話成立で自動停止して結果入力へ
- 結果保存後は次番号がセットされ、ワンタップで次発信できる状態に

完全自動の連続発信が必要な場合は、TFTアプリ（https://tft-app.com/ ）等の
ネイティブアプリとの併用をご検討ください。

## デプロイ手順

### Step 1: GASを再デプロイ (必須)
1. Apps Script エディタ（hr.ashikaga@gmail.com アカウント）を開く
2. `gas-code.txt` の内容を全コピーして既存コードに上書き
3. **「setupSheets」を一度実行**（「発信件数」シートを作成するため）
4. 「デプロイ」→「デプロイを管理」→「編集」→「新しいバージョン」→「デプロイ」

⚠️ 発信件数通知の送信元を hr.ashikaga@gmail.com にするには、
このGASプロジェクトが hr.ashikaga@gmail.com アカウントで作成・実行されている必要があります。

### Step 2: GitHub に7ファイルアップロード
`.nojekyll` / `index.html` (UTF-8 BOM付き, v26) / `gas-code.txt` / `manifest.json` / `sw.js` (v26.0.0) / `icon-192.png` / `icon-512.png`

### Step 3: 全端末でブラウザキャッシュ完全クリア
- PC (Chrome): F12 → Application → Storage → Clear site data → Ctrl+Shift+R
- スマホ: PWAアイコン削除 → ブラウザでサイトデータクリア → 再度ホーム画面追加

### Step 4: 動作確認
1. 管理者 (PIN 9999) でログイン → 「リスト読込」に「藪内亜希」が選択肢に出る
2. 藪内に5件アップロード
3. 別端末で PIN 097 ログイン → 5件見える
4. 「⚡ オートコール」をON → 応答なしで自動で次へ進むか確認
5. 退勤 → tereapo01.sukima@gmail.com に「発信件数」メールが届くか確認

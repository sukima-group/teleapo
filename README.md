## デプロイ手順

### Step 1: GAS を更新
1. **hr.ashikaga@gmail.com** でログインし https://script.google.com/ を開く
2. 既存「テレアポ管理」プロジェクトの「コード.gs」を全削除
3. `gas-code.txt` の内容を全コピー → 貼り付け → 保存
4. 関数プルダウンで **「setupSheets」を選択 → 実行**（「発信件数」シート作成）
5. 「デプロイ」→「デプロイを管理」→「編集」→「新しいバージョン」→「デプロイ」

> ⚠️ 発信件数メールの送信元を hr.ashikaga にするには、GASプロジェクトが hr.ashikaga アカウントで動いている必要があります。

### Step 2: GitHub に7ファイルアップロード
1. https://github.com/sukima-group/teleapo を開く
2. 「Add file」→「Upload files」
3. 上記7ファイル（+README）をアップロード
4. 「Commit changes」

### Step 3: 全端末でキャッシュクリア（最重要）

**PC (Chrome)**:
1. F12 → Application → Storage → **Clear site data**
2. Service Workers → **Unregister**
3. **Ctrl+Shift+R**

**スマホ**:
1. PWAアイコンを**長押し → 削除**
2. Chrome → 設定 → サイトの設定 → サイトデータを削除
3. 再アクセス → 「ホーム画面に追加」

### Step 4: 動作確認
- ✅ ログイン画面に「**テレアポ管理 v26**」と表示される
- ✅ PIN **097** で「**藪内亜希**」が表示される
- ✅ 発信ボタンの上に「**⚡ オートコール（連続発信）**」トグルがある
- ✅ 退勤時に tereapo01.sukima@gmail.com へ「**【発信件数】**」メールが届く

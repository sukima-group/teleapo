## 🆕 v35 の変更点

### ① 会社ロゴの反映＋「大手企業の独自システム」風デザイン
アポインターが**信頼感**を持って使えるよう、スキマのロゴを全面に反映し、企業システムらしい落ち着いた配色（ネイビー基調＋ロゴのオレンジをアクセント）に刷新しました。

- **PWAアイコン**（ホーム画面）を**スキマのロゴ**に差し替え（192px / 512px）
- **起動スプラッシュ**：ロゴ＋「川嶋メソッド／テレアポ業務管理システム」＋ローディングアニメーション
- **ログイン画面**：ロゴを大きく配置し、`🔒 暗号化された安全な接続`・`© 2026 スキマグループ合同会社` を表示
- **全画面共通ヘッダー**：左上にロゴを常設し、システム名「川嶋メソッド」を明示
- ロゴは**画像を本体に埋め込み（Base64）**しているため、オフラインでも確実に表示されます

### ② お知らせ機能（経営層 → アポインター）※信頼感を高める新機能
大手の社内ポータルのように、**経営層からのお知らせをアプリ内に掲示**できるようにしました。

- **管理者**：タイトル・本文・重要度（通常／重要）を入力して「掲載」。投稿一覧から**掲載/非掲載の切替・削除**が可能
- **アポインター**：ホーム画面の最上部に**お知らせバナー**を表示。**重要**は赤で強調され、各自「既読」を押すまで目立つ形で残ります
- お知らせはサーバー（新シート「お知らせ」）で一元管理され、全アポインターに即共有されます

## デプロイ手順

### Step 1: GAS を更新（お知らせ機能のため必須）
1. hr.ashikaga@gmail.com でログインし https://script.google.com/ を開く
2. 「コード.gs」を **Ctrl+A → Delete** → `gas-code.txt` を貼り付け → **Ctrl+S**
3. 「デプロイ」→「デプロイを管理」→「✏️編集」→「**新しいバージョン**」→「デプロイ」

### Step 2: GitHub に8ファイルアップロード
https://github.com/sukima-group/teleapo →「Add file」→「Upload files」→ **8ファイルすべて**（特に icon-192.png / icon-512.png を忘れずに）→「Commit changes」

### Step 3: 全端末でキャッシュクリア＆アイコン再追加
- **PC**: F12 → Application → Clear site data → Service Workers → Unregister → Ctrl+Shift+R
- **スマホ**: PWAアイコン長押し→削除 → サイトデータ削除 → 再アクセス → **ホーム画面に追加（ロゴアイコンになります）**

### Step 4: 動作確認
- ✅ ホーム画面アイコン・起動スプラッシュ・ログイン画面・ヘッダーに**スキマのロゴ**が表示される
- ✅ 管理者「📢 お知らせ」で投稿 → アポインターのホーム最上部にバナー表示、「既読」で消える
- ✅ 重要お知らせが赤色で強調される
- ✅ 既存機能（15分刻みシフト／再架電保存／シフトExcel・承認／架電実績Excel／面談予約）も従来どおり動く
